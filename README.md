# use_redland
Hello World application for using Redland

## Install Dependencies
- Install librdf:
  - `$ sudo apt install librdf0-dev`
- Install Rasqal
  - `$ sudo apt install librasqal3-dev`
- Install Raptor
  - `$ sudo apt install libraptor2-dev`

## Build with CMake
- `$ mkdir build; cd build`
- `$ cmake ..`
- `$ make`
